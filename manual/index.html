How to install an SPT program on a PC

​







System Requirements:

  • 80mb disk space
  • 2GB ram
  • 64bit OS architecture (Windows and android versions also support 32bit OS architectures)
  • It is recommanded to have a network connection

List of compatible operating systems

  • Windows 7*, 8 or 10 (32bit and 64bit)
  • macOS 10.9 or later (64bit)
  • Debian Linux (64bit)
  • Android 2.0 or later (32bit and 64bit)

* Windows 7 requires SP1 to be installed

​

1. Install on a PC running Windows 8 or 10
  1. Click on the download link.
  2. Run the file.
  3. A SmartScreen pop-up will appear telling you that this is not a trusted developer. Click on "More info..." and then on "Run anyway".
  4. Follow the on-screen instructions




2. Install on a PC running Windows 7


  1. Click on the download link.
  2. Your browser may advert you that the file may be dangerous. Just click on"Download anyway".
  3. Run the file.
  4. A Windows firewall pop-up will appear telling you that this is not a trusted developer. Just click on "Run ".
  5. Follow the on-screen instructions




3. Install on a Mac running macOS 10.9 or +​


  1. Click on the download link.
  2. Your browser will advert you that the file may be dangerous. Just click on"Download anyway".
  3. If you have a package, please install the package. ALERT!!!: If there are more than one package please install ALL the packages!. If you have an application, directly drag the app to the applications folder.
  4. Run the application. A system preferences pop-up may tell you that this in not a trusted developer. Click "OK".
  5. Open Applications folder, Control+Click on the app and click on "Open".




4. Install on a device running android


  1. Click on the download link.
  2. Open the extrackted .apk file
  3. Enable applications from unknown sources.
  4. Google Play Store may advert you that this is not a secure app. Click on Install anyway.
  5. Run the application and enjoy!




5. Install on a device running Debian based Linux


This is for all the debian-based linux systems, such as Debian, Ubuntu, Ubuntu Mate, Kubuntu, Lubuntu, Rubuntu, Linux Mint, ElementaryOS, Kali linux, etc.

  1. Click on the download link.
  2. Open the folder where the downloaded file is placed
  3. Install the .deb file as you would open a normal file.
  4. Follow the on-screen instructions.

    5. NOTE: If you don't have any application to open .deb files, follow the steps below:

    1. Open a new terminal and type the following code:

      sudo apt install gdebi

      You'll need to write your password.

    2. Now you can open the downloaded file as normal. Just open the file and follow the on-screen instructions.
