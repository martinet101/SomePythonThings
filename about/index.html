What is SomePythonThings?

SomePythonThings Is a personal project made without lucrative pruposals and only for learning. Those programs are free, you can download it (if not, why the download buttons), and feel free to recommend this program if it was useful :D



How is all of this made?

All of the python programs are made purely with Python and Tkinter

The text editor and the calc for mac are made with Python, Tkinter and tkmacosx

The iTunes launcher just runs via shell script the iTunes source code wich contains inside

All android applications are made with MIT Appinvenor (ai2.appinventor.mit.edu), and some of them have a web-based engine such as SPT Calc.
