How to install an SPT program on a PC

​

It is requiered a 64bit OS!!!

​






1. Install on a PC running Windows 8, 8.1 or 10 (64bit)
  1. Click on the download link.
  2. Your browser may advert you that the file may be dangerous. Just click on"Download anyway".
  3. Run the file.
  4. A SmartScreen pop-up will appear telling you that this fis not a trusted developer. Click on "More info..." and then on "Run anyway".
  5. Install the program and enjoy.




2. Install on a PC running Windows XP, Vista or 7 (64bit)


  1. Download and install any Python 3 version
  2. Click on the download link.
  3. Your browser may advert you that the file may be dangerous. Just click on"Download anyway".
  4. Run the file.
  5. A Windows firewall pop-up will appear telling you that this is not a trusted developer. Just click on "Run ".
  6. Install the program and enjoy.




3. Install on a Mac running macOS 10.9 or +​


  1. Click on the download link.
  2. Your browser will advert you that the file may be dangerous. Just click on"Download anyway".
  3. Install the package.
  4. Run the application. A system preferences pop-up may tell you that this in not a trusted developer. Click "OK".
  5. Open Applications folder, Control+Click on the app and click on "Open".




4. Install on a device running android


  1. Click on the download link.
  2. Open the extrackted .apk file
  3. Enable applications from unknown sources.
  4. Google Play Store will advert you that this is not a secure app. Click on Install anyway.
  5. Run the application and enjoy!
